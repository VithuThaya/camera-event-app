# Screenshots

The project README shows three phone screenshots from this folder. Drop your own
in here — the README picks them up **automatically** as soon as files exist with
these exact names (all lowercase):

- `join.png` — a guest joining by scanning the event QR code (or the consent screen)
- `capture.png` — the capture screen: viewfinder, shutter, shots-left counter
- `slideshow.png` — the host running the roll as a slideshow after the reveal

Tips:

- Use **portrait** phone screenshots — they sit side by side at 240px wide.
- PNG or JPG both work; keep the `.png` names, or update the paths in the root
  `README.md` if you use a different extension.
- On iPhone: press **Side + Volume Up** to screenshot, then AirDrop or save the
  image to your Mac and copy it into this folder.

Until you add them, the README simply shows three broken-image placeholders —
harmless, and they disappear the moment the files land here.
